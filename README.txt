README
Date: November 22, 2019

Replication files and code for Oremus, "Climate variability reduces employment in New England fisheries", PNAS, 2019. 

SOFTWARE REQUIRED:
STATA v15 (require user to install coefplot.ado, ranktest.ado, estout.ado, lincomest.ado, reghdfe.ado, ftools.ado, and ivreg2.ado)
MATLAB R2015a

SCRIPTS:
STATA: generateFigure_1a_2.do, generateFigure_3.do
R: generateFigure_1b.R

FILE STRUCTURE:
/code: contains all Stata and R scripts
/input: contains all input files
/figures: for all output figures
/temp: for all temporary files

INSTRUCTIONS:
Run generateFigure_1_2.do, generateFigure_3.do and generageFigure_1b.R to generate all figures in the main text.
